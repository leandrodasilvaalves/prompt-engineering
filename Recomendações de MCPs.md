## Recomendações de MCPs

Lista de MCP servers recomendados para uso com agentes de IA em contextos de DevOps e desenvolvimento.

| MCP Server | Descrição |
|---|---|
| `github-mcp-server` | Integração com GitHub — leitura de repos, issues, PRs e código |
| `context7` | Busca de documentação atualizada de bibliotecas e frameworks |
| `gitmcp.io` | Transforma qualquer repositório Git em fonte de contexto para o agente |
| `microsoft/playwright-mcp` | Automação de browser — permite ao agente navegar e interagir com páginas web |
| `mcp-server-kubernetes` | Gerenciamento de clusters Kubernetes via agente |
| `pab1it0/prometheus-mcp-server` | Consulta de métricas e alertas do Prometheus |
| `browsermcp.io` | Controle de browser para automação e scraping |
| `terraform-mcp-server` | Gerenciamento de infraestrutura Terraform via agente |
